Overview

Purpose

The purpose of this website is to provide information about the Remind Mi service, and how it can help students and people in general manage their time and goals more efficiently

Audience

College Students / People trying to manage their time

Dynamic elements

  • Sticky Header: Stays or changes color as the user scrolls
  • Form Validation: validate to do list
  • Dynamic To-DO list: Mock up of to-do list with priority levels that update dynamically
    1. I will store task data in a JavaScript array of objects and use a forEach loop to render them into the DOM. I will use event listeners on filter buttons to update the display without a page reload.
